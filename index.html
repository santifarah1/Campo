Margen por actividad

Ingresos menos costos directos de cada actividad. Los costos de estructura (sueldos, combustible, bancarios) van aparte porque no son de una sola actividad — así ves la rentabilidad real de cada negocio.

Activá el asesor

El asesor usa Gemini de Google (gratis) para razonar sobre los datos del campo. Necesitás una clave gratuita, se pega una sola vez y queda guardada en esta PC. Tus datos no se suben a ningún lado — solo se manda un resumen de números cuando hacés una pregunta.

Cómo sacar la clave (2 min):
1. Entrá a aistudio.google.com/apikey con tu cuenta de Google.
2. Tocá "Create API key" / "Crear clave".
3. Copiala y pegala acá abajo.
